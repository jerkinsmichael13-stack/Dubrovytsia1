⚙️ Налаштування GitHub

Одноразове налаштування! Вам потрібен GitHub Personal Access Token, щоб адмін-панель могла зберігати фото безпосередньо на GitHub без ручного редагування коду.
Як отримати токен (1 хвилина):

Перейдіть на github.com/settings/tokens/new

У полі "Note" напишіть: Дубровиця архів

У "Expiration" оберіть "No expiration"

Поставте галочку "repo" (перший пункт у списку)

Натисніть зелену кнопку "Generate token"

Скопіюйте токен (він починається з ghp_) і вставте нижче

✅ Токен збережено. GitHub API підключено.

Додати фото до архіву

Як завантажити фото: Перейдіть на imgur.com/upload, завантажте фото, натисніть правою кнопкою → "Копіювати адресу зображення". Для оригіналу завантажте файл на Google Drive і скопіюйте посилання для перегляду.
💡 Для завантаження важких файлів. У Google Drive: ПКМ на файлі → Отримати посилання → Копіювати.
